# PRF Review — Download & Install (field techs)

**Bookmark this one link. It always gives you the newest, tested version.**

## Download
**Installer (most people):**
https://github.com/lakeosoyoos/prf-review-app/releases/download/windows-build/PRF-Review-Setup.exe

**Portable ZIP (locked‑down PCs that block installers):**
https://github.com/lakeosoyoos/prf-review-app/releases/download/windows-build/PRF-Review-Windows.zip

## Install (about 30 seconds)
1. Click the **installer** link above → it downloads `PRF-Review-Setup.exe`.
2. Double‑click the downloaded file.
3. **If Windows shows a blue “Windows protected your PC” box:** click **More info** → **Run anyway**.
   *(That warning just means the app isn’t code‑signed yet — it’s expected.)*
4. Follow the prompts. A **PRF Review** shortcut lands on your Desktop and Start menu.

*Portable ZIP instead:* download the ZIP → right‑click → **Extract All** → run **PRF Review.exe** from the
extracted folder. (No install; good for machines that block setup files.)

## To update later
Just click the **same bookmarked link** again and run it. It installs over the old one. **No version
hunting — the link is always the current build.**

## Found a problem?
- Open the app and look at the top — it shows **“Build ####”**. Include that number.
- Send the build number + a short description (and a screenshot if you can) to: **your office reviewer**.
